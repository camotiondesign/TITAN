# LinkedIn Assets

LinkedIn-specific social media content.

## Carousels

- **carousel-1/**: First carousel post (images + caption)
- **carousel-2/**: Second carousel post (images + caption)

Each carousel folder should contain:
- Individual slide images
- Caption/copy file (markdown or text)

## Single Images

- **single-image-1/**: First single image post
- **single-image-2/**: Second single image post
- **single-image-3/**: Third single image post
- **single-image-4/**: Fourth single image post

Each single image folder should contain:
- The image file
- Caption file (markdown or text)

## Caption Guidelines

Follow the 3-line hook structure:
- Line 1: Specific statement
- Line 2: Break
- Line 3: Twist/reveal/question

See Titan AI Instructions for full caption framework.
