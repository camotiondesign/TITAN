# Blog Template Structure

This template shows the folder structure for organizing blog content and assets.

## Structure

```
blog-title/
├── blog/
│   └── [blog-title].md
├── research/
│   └── [research notes, sources, data]
├── supporting-assets/
│   └── [images, charts, diagrams, etc.]
└── socials/
    └── linkedin/
        ├── carousel-1/
        ├── carousel-2/
        ├── single-image-1/
        ├── single-image-2/
        ├── single-image-3/
        └── single-image-4/
```

## Usage

1. Copy this template folder
2. Rename it to match your blog title (sanitized, lowercase, hyphens)
3. Add your blog markdown file to `blog/`
4. Add research notes and sources to `research/`
5. Add supporting visuals to `supporting-assets/`
6. Create LinkedIn assets in `socials/linkedin/` subfolders

## Folder Descriptions

- **blog/**: The main blog post markdown file
- **research/**: Research notes, sources, data, transcripts, etc.
- **supporting-assets/**: Images, charts, diagrams, screenshots used in the blog
- **socials/linkedin/**: LinkedIn-specific social media assets
  - **carousel-1/**, **carousel-2/**: Carousel post assets (images, copy)
  - **single-image-1/**, **single-image-2/**, etc.: Single image post assets (image, caption)
