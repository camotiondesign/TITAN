# Carousel Design Prompt: POV: Running a Service-Led Health Hub

## Campaign Context
- **Campaign Folder:** `/campaigns/TITANVERSE/2025-12-04-from-dispensing-to-health-hub/`
- **Content Source:** `content/carousel/slides.md`
- **Objective:** Demonstrate the shift from dispensing-led to service-led health hub model, showing how Titanverse enables this transformation
- **Target Audience:** Pharmacy owners, Superintendents, Pharmacists considering service expansion, Industry observers
- **Campaign Type:** Thought Leadership / Product Education
- **Brand:** TITANVERSE

## Design Reference
- **Visual Style:** Reference `/design-reference/titanverse-visual-style.md`
- **Carousel Patterns:** Reference `/design-reference/carousel-layout-patterns.md`
- **Brand Colors:** Reference `/design-reference/brand-colors-quick-reference.md`
  - **Primary Gradient:** `#FF65DD` (pink) → `#B752FF` (purple)
  - **Supporting:** White, Black
- **Top Performer Examples:** 
  - POV Start Consultation (76.8% engagement, 73.8% CTR) - POV perspective, product UI
  - AI Templates (58.0% engagement) - Visual metaphor progression
  - Your Pharmacy Isn't Struggling (53.4% engagement) - Text-focused with minimal visuals

## Important: Avoid Side-by-Side Before/After
**This visual device is overused.** Use one of these fresh alternatives instead:
- POV perspective (shows product in use)
- Single powerful visual (focuses on outcome)
- Visual metaphor progression (transformation narrative)
- Text-focused with minimal visuals (typography as hero)

## Slide-by-Slide Specifications

### Slide 1 – Hook: "Two pharmacies. Same building. Very different future."

**Copy:**
```
Two pharmacies. Same building.

Very different future.
```

**Visual Style Options (Choose ONE - avoid side-by-side):**

**OPTION 1: POV Perspective (RECOMMENDED - Highest Engagement)**
- **Layout:** First-person perspective showing hands holding tablet/device
- **Visual Elements:**
  - Hands holding glowing tablet displaying Titanverse console
  - Morning schedule view visible on screen:
    - "9:00 AM - NMS" (filled slot)
    - "10:30 AM - Travel Health" (filled slot)
    - "2:00 PM - IP Clinic" (filled slot)
    - "3:30 PM - Private Service" (filled slot)
  - Clean, organized digital interface
  - Titanverse purple console UI theme (`#FF65DD` → `#B752FF` gradient)
- **Headline:** "Two pharmacies. Same building. Very different future." in bold, large typography
  - White text on dark purple gradient background
  - Or black text on light background
- **Background:** Dark purple-to-black gradient (Titanverse cosmic theme) or clean white
- **Why It Works:** Shows the "new" model directly, creates immediate relatability, demonstrates product in use
- **Reference:** POV Consultation carousel (76.8% engagement, 73.8% CTR)

**OPTION 2: Single Powerful Visual (Calendar/Schedule Focus)**
- **Layout:** Large calendar/schedule interface as hero visual
- **Visual Elements:**
  - Full day of appointments visible (NMS, Travel Health, IP Clinics, Private Services)
  - No dispensing queue visible (emphasizing shift)
  - Clean, organized, service-focused layout
  - Titanverse purple console UI (`#FF65DD` → `#B752FF` gradient)
- **Headline:** Overlaid or positioned prominently
- **Background:** Clean white or dark purple gradient
- **Why It Works:** Focuses on transformation outcome, single visual is more impactful than comparison

**OPTION 3: Visual Metaphor - Transformation Flow**
- **Layout:** Diagonal transformation visual
- **Visual Elements:**
  - Bottom-left: Faded, chaotic dispensary elements (prescriptions, baskets, paper) in muted colors
  - Top-right: Clean, organized service schedule/console in vibrant Titanverse purple gradient
  - Diagonal flow suggests transformation/progression
  - More dynamic than static side-by-side
- **Headline:** Positioned to guide eye along transformation
- **Background:** Gradient from muted (bottom-left) to vibrant purple (top-right)
- **Why It Works:** Shows transformation without cliché side-by-side, more engaging flow
- **Reference:** Similar to "From Noise to Calm" transformation pattern

**Typography:**
- Headline: Bold, large, white (on dark) or black (on light)
- Clean, readable sans-serif throughout

**Technical:**
- Dimensions: 1080x1080px
- Format: PNG
- File naming: `2025-12-07-pov-service-led-health-hub-slide-01.png`

### Slide 2 – Problem/Pain: "Dispensing alone is no longer sustainable."

**Copy:**
```
Dispensing alone is no longer sustainable.

"We are not chasing items anymore. We are about service."
```

**Visual Style Options (Choose ONE - avoid side-by-side):**

**OPTION 1: Problem Focus (RECOMMENDED - Creates Tension)**
- **Layout:** Single powerful visual focusing on the problem
- **Visual Elements:**
  - Chaotic dispensary scene:
    - Stressed pharmacist buried in baskets of prescriptions stacked high
    - Paper forms scattered everywhere
    - Manual processes visible (printing, filing)
    - Queue of patients waiting
  - **Red color coding:**
    - Red X marks indicating problems
    - Red accents on chaotic elements
  - Background: Light grey/white with red problem indicators
- **Quote Text:**
  - Displayed prominently
  - "We are not chasing items anymore" in faded/grey
  - "We are about service" in bold, highlighted Titanverse purple gradient (`#FF65DD` → `#B752FF`)
- **Why It Works:** Shows problem first, solution comes later - creates narrative tension
- **Reference:** AI Templates carousel (58.0% engagement) - shows problem first, solution later

**OPTION 2: Visual Metaphor (Stacks/Overwhelmed)**
- **Layout:** Stacks of prescription baskets, paper forms, manual processes piled high
- **Visual Elements:**
  - Overwhelmed visual metaphor
  - Red color coding for problems
  - Quote overlaid or positioned prominently
- **Background:** Clean, focused on the problem
- **Why It Works:** Visual metaphor is memorable, no solution shown yet - creates curiosity

**OPTION 3: Text-Focused (Typography as Hero)**
- **Layout:** Strong typography as primary visual
- **Visual Elements:**
  - Large, bold quote text:
    - "We are not chasing items anymore" in faded/grey
    - "We are about service" in bold, highlighted Titanverse purple gradient
  - Minimal visual elements (maybe subtle chaotic background pattern or prescription icon)
  - Clean, professional design
- **Background:** Clean white or light gradient
- **Why It Works:** Typography creates impact, minimal visuals keep focus on message
- **Reference:** Your Pharmacy Isn't Struggling carousel (53.4% engagement)

**Typography:**
- Main headline: Bold, large, black
- Quote: Large, readable, with color contrast for emphasis
- Clean sans-serif throughout

**Technical:**
- Dimensions: 1080x1080px
- Format: PNG
- File naming: `2025-12-07-pov-service-led-health-hub-slide-02.png`

- **Alt Text:** Each slide has detailed alt text in `content/carousel/slides.md`
- **Color Independence:** Don't rely solely on color to convey meaning (use icons, text, shapes)

## Design Principles

### Visual Hierarchy
1. **Hook Slide:** POV perspective or single powerful visual (NOT side-by-side)
2. **Problem Slide:** Problem-focused (no solution shown yet)
3. **Structure/Workflow Slides:** Product UI, clean visuals, organized
4. **Proof Slide:** Clean quote, minimal design
5. **CTA Slide:** Focused, clear action

### Brand Consistency
- **Titanverse Purple Gradient:** `#FF65DD` → `#B752FF` used consistently
- **Typography:** Clean, modern sans-serif throughout
- **Layout:** Clean, uncluttered, professional
- **Avoid:** Side-by-side before/after (overused device)

### Narrative Flow
- **Clear Progression:** Each slide builds on previous
- **Visual Consistency:** Maintain style throughout
- **Progressive Revelation:** Build understanding slide by slide
- **Strong Contrast:** Hook slide must stop scroll (without cliché side-by-side)
