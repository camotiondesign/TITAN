# Brand Colors Quick Reference

## Titanverse Brand Colors

### Primary Gradient
- **Gradient**: `#FF65DD` (bright pink/magenta) → `#B752FF` (purple)
- **Usage**: Primary brand identity, backgrounds, text highlights, icons, stars, sparkles
- **Direction**: Can be used vertically, horizontally, or diagonally
- **Examples**: 
  - Logo text ("VERSE" in gradient)
  - Quote highlights ("The AI is literally" in pink)
  - Sparkle icons (purple-to-pink gradient)
  - Star elements
  - Background gradients (dark purple-to-black)

### Supporting Colors
- **White**: Text on dark backgrounds, clean elements
- **Black**: Strong contrast, emphasis
- **Deep Purple/Black**: Dark backgrounds for cosmic theme

---

## Titan PMR Brand Colors

### Primary Gradient
- **Gradient**: `#2DA7FF` (light blue) → `#2D40FF` (darker blue)
- **Usage**: Primary brand identity, backgrounds, key visual elements
- **Direction**: Can be used vertically, horizontally, or diagonally
- **Examples**:
  - Brand identity elements
  - Background gradients
  - Text highlights
  - Key visual elements

### Supporting Colors
- **White**: Clean backgrounds, text, documents
- **Black**: Typography, text elements, buttons, strong contrast
- **Light Grey/Blue**: Subtle backgrounds, grid patterns, clinical aesthetic
- **Red**: For problems/pain points (empty batteries, X marks)
- **Green**: For solutions/positive outcomes (full batteries, checkmarks)

---

## Usage Rules

### Do
- Use Titanverse gradient (`#FF65DD` → `#B752FF`) for all Titanverse content
- Use Titan gradient (`#2DA7FF` → `#2D40FF`) for all Titan PMR content
- Use black and white where suitable for both brands
- Maintain gradient direction consistency within each asset

### Don't
- Mix Titan blue gradient in Titanverse content
- Mix Titanverse purple gradient in Titan PMR content
- Use gradients in opposite brand contexts
- Deviate from specified hex codes
