# Titan PMR Visual Style Guide

## Overview

Titan PMR uses a clean, professional, clinical aesthetic with a blue gradient color palette. Visuals emphasize efficiency, reliability, and operational excellence in pharmacy management.

---

## Color Palette

### Primary Brand Gradient
- **Titan Blue Gradient**: `#2DA7FF` (light blue) to `#2D40FF` (darker blue)
- **Usage**: Primary brand identity, backgrounds, key visual elements
- **Direction**: Can be used vertically, horizontally, or diagonally

### Supporting Colors
- **White**: Clean backgrounds, text, clothing elements, documents
- **Black**: Typography, text elements, buttons, strong contrast
- **Light Grey/Blue**: Subtle backgrounds, grid patterns, clinical aesthetic

### Color Usage
- **Backgrounds**: 
  - White (most common, clean and professional)
  - Light blue grid pattern (technical, clinical feel)
  - Blue gradient (for emphasis, hero sections)
- **Text**: 
  - Black (primary text, headlines)
  - White (on dark/blue backgrounds)
  - Blue gradient (for emphasis, brand elements)
- **Visual Elements**: 
  - Blue gradient for primary elements
  - Black for strong contrast and CTAs
  - White for clean, clinical feel

---

## Typography

### Headlines
- **Font**: Bold, sans-serif
- **Style**: Uppercase for emphasis, title case for readability
- **Color**: Black or white (depending on background)
- **Size**: Large, prominent, impactful

### Body Text
- **Font**: Clean sans-serif
- **Style**: Readable, professional
- **Color**: Black or dark grey on light backgrounds, white on dark backgrounds
- **Size**: Standard, readable

### Brand Text
- **"TITAN"**: Bold, black, uppercase sans-serif
- **Logo**: Can appear in black or white depending on background

## Layout Patterns

### Problem-First Layout (High Performance)
- **Structure**: Strong visual metaphor for problem (broken elements, empty batteries, chaos)
- **Colors**: Red for problems, blue for solutions
- **Visual Elements**: 
  - Broken question marks (electric blue, faceted/geometric)
  - Empty battery icons (red for depleted resources)
  - Prescription pads, Rx symbols
  - Grid backgrounds (clinical, technical)
- **Use Case**: Problem identification, pain point communication
- **Example**: "Why Independent Prescribing is Failing" - broken question mark over prescription pad

### Resource Visualization
- **Structure**: Battery icons representing Time, Energy, Money
- **Colors**: 
  - Green batteries (full resources)
  - Red batteries (empty/depleted resources)
- **Visual Elements**: 
  - Character illustrations (friendly, simplified cartoon style)
  - Battery icons with 4 segments
  - Clear labels (Time, Energy, Money)
- **Use Case**: Pain point visualization, resource scarcity communication
- **Example**: "Pharmacist" with all empty batteries (Time, Energy, Money all red)

### Testimonial/Case Study Layout
- **Structure**: Real photo + bold statistics overlay
- **Colors**: 
  - White background (most common)
  - Blue text for statistics
  - Black text for emphasis
- **Visual Elements**: 
  - Real pharmacy photos
  - Real people (pharmacists, team members)
  - Bold statistics (large numbers, blue text)
  - Pharmacy signage visible
  - "Swipe & See how" with pointing finger emoji (for carousels)
- **Use Case**: Case studies, testimonials, real-world usage
- **Example**: "170K ITEMS. 12 PHARMACIES. 0 CHAOS." with pharmacist photo

### Blog Cover Layout
- **Structure**: Title + illustration + CTA button
- **Colors**: 
  - Blue gradient background
  - White text
  - Black CTA button
- **Visual Elements**: 
  - Stylized illustrations (hand writing, geometric shapes)
  - Abstract blue geometric backgrounds
  - Halftone dot patterns
  - Clean typography
- **Use Case**: Blog post headers, article thumbnails
- **Example**: "Why Independent Prescribing Fails" with hand writing illustration

### Event Promotion Layout
- **Structure**: Speaker photo + event details + branding
- **Colors**: 
  - Blue gradient background
  - White header section
  - Black text for titles
  - Green accents (event branding)
- **Visual Elements**: 
  - Real speaker photos (warm, approachable)
  - Event logos and dates
  - Lanyards with brand logos
  - Clean, professional layout
- **Use Case**: Event promotion, speaker features, conference materials
- **Example**: "Inside Tariq's Pharmacy Show Talk" with speaker photo

## Usage Guidelines

### When to Use This Style

**Titan PMR Brand:**
- Operational efficiency content
- Problem-first narratives
- Case studies and testimonials
- Resource management messaging
- Pharmacy operations content

**Visual Applications:**
- Problem identification graphics
- Case study visuals
- Resource visualization
- Blog covers
- Event promotion

### What Works

- **Blue Gradient**: Use for brand identity, backgrounds, key elements
- **White Backgrounds**: Clean, professional, most common
- **Real Photos**: Authentic, trustworthy, relatable
- **Bold Statistics**: Large numbers, blue text, high contrast
- **Grid Patterns**: Technical, clinical aesthetic
- **Problem-First Visuals**: Broken elements, empty resources, visual metaphors

### What to Avoid

- **Too Many Colors**: Stick to blue gradient, black, white
- **Cluttered Layouts**: Keep it clean and focused
- **Dark Backgrounds**: Prefer white/light backgrounds (except for specific emphasis)
- **Overly Stylized**: Maintain professional, approachable aesthetic
- **Complex Illustrations**: Keep simple and clear

## Technical Specifications

### File Formats
- **Illustrations**: Vector (SVG, AI) preferred, high-res PNG for raster
- **Photography**: High-resolution JPG/PNG
- **Final Export**: PNG for social media, maintain source files

### Dimensions
- **Carousel Slides**: 1080x1080px (LinkedIn)
- **Single Images**: Platform-specific (LinkedIn: 1200x627px, etc.)
- **Blog Covers**: 1200x630px (standard blog thumbnail)
- **Event Posters**: Varies by use case

### Accessibility
- **Contrast**: Ensure text meets WCAG AA standards
- **Alt Text**: Always provide detailed alt text descriptions
- **Color Independence**: Don't rely solely on color to convey meaning
