# Where to start for performance analysis

1. **Overview:** `posts/_master-index.md` — cross-brand summary, top performers, recent posts.
2. **Brand-level detail:** `posts/linkedin/titan/published/_index.md` or `posts/linkedin/titanverse/published/_index.md` if needed.
3. **Short metrics summary:** `analytics/linkedin-metrics-summary.md` (if present).
4. **Full post-level data:** `analytics/aggregated-linkedin-metrics.json` — use only for bulk or query-style analysis; prefer the indexes above for verdicts.

Do not read the full `data/notion/notion_export.json` or `analytics/aggregated-linkedin-metrics.json` unless you are bulk querying.
