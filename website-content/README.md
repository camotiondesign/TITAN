# Website content

Canonical copies of published Titan website content for reference, tagging, and reuse (social, email, campaigns).

## Folders

| Folder | Description |
|--------|-------------|
| **blogs** | Published blog posts (titanpmr.com/blog/...). Each has `meta.json` + `content.md`. |
| **case-studies** | Pharmacy case studies (e.g. Safy's Chemist). Each has `meta.json` + `content.md`, with stats and outcomes. |
