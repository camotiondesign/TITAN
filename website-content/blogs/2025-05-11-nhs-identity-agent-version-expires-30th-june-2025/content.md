# NHS Identity Agent version updates

The NHS manage the Identity Agent software version lifecycle. To keep up to date with the latest versions along with any software end of life notices, refer to this NHS page: https://digital.nhs.uk/services/care-identity-service/releases-and-planned-maintenance/service-deprecation-and-retirement/software-and-hardware-retirement

This is an important notification that versions 2.4.6.0 and earlier of the NHS Identity Agent will stop working after 31st March 2026.

To help you with this upgrade, we have built a simple tool to allow you to update the Identity Agent on any PCs you have that are running an older version:

[Download and run NHS Identity Agent upgrade tool](https://invatechhealth.com/IAUpgrade.zip)

1. Click the link to download
2. Visit your Downloads folder
3. Right click to unzip the IA checker
4. Double click to run

The tool will automatically check your installed version and upgrade it if necessary.

Please note: Local administrator rights are required to run this tool.

We recommend completing this update as soon as possible to avoid any disruption to your NHS services.
