## Blog Format Rules

- **Length:** 800–1,200 words. Pharmacists scan, not read.
- **No fluff intro.** Start with stat or problem. Never "As the healthcare landscape evolves..."
- **One real named example per post** (pharmacy name + pharmacist name + numbers)
- **One CTA at the end.** Not mid-article.
- **H2 headings only.** Short and active.
- **SEO:** Target keyword in H1 + first paragraph. No stuffing.
- **Internal links:** One case study link + one product page link per blog.
