# Legacy PMRs Print Everything

Animated loop video positioning legacy PMRs as outdated by highlighting excessive printing and chaos. Visual format effectively communicated contrast between legacy and modern digital approaches.

## Primary Asset

- Animated loop video (shortform)

## Supporting Assets

- LinkedIn post

## Strategic Note

Visual animation format effectively communicated legacy PMR pain points (excessive printing, chaos, outdated behavior). Comments from Aron Garner and Balal Anwar validated innovation positioning.

---

## Campaign Overview

**Published:** February 19, 2025  
**Product:** Titan PMR  
**Type:** Brand Positioning  
**Purpose:** Position legacy PMRs as outdated by highlighting excessive printing and contrast with modern digital approach.

## Core Message

Some PMRs still brag about printing prescriptions automatically—that's like celebrating fax machines in 2025. Pharmacy has evolved. Legacy PMRs haven't.
