# 2025-07-01 Brighton Hill Case Study

**Case Study: How Brighton Hill Pharmacy Scaled from 4K to 17K Without the Stress**

Brighton Hill Pharmacy case study featuring Sagar Lungiwala's transformation of a former Boots branch (4,000 items/month) into a high-volume, service-led pharmacy (17,000+ items/month) using Titan PMR and Meditech robot integration.

## Campaign Overview

**Published:** July 1, 2025  
**Product:** Titan PMR  
**Type:** Case Study  
**Subject:** Brighton Hill Pharmacy, Basingstoke (Sagar Lungiwala)

## Key Metrics

- 4× item growth (4,000 → 17,000+ items/month)
- 80% of time freed for services
- 100% paperless workflow
- First UK pharmacy to integrate Titan PMR with Meditech robot
