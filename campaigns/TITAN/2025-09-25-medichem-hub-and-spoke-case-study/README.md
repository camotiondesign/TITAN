# 2025-09-25 Medichem Hub-and-Spoke Case Study

**Case Study: How Medichem Built a Hub-and-Spoke Network with Titan**

Medichem Pharmacy case study featuring Jeet's transformation of a four-branch network across West Yorkshire, implementing hub-and-spoke model with Titan P2P, achieving 4x dispenser efficiency and 45% stockholding reduction.

## Campaign Overview

**Published:** September 25, 2025  
**Product:** Titan PMR  
**Type:** Case Study  
**Subject:** Medichem Pharmacy, Leeds and West Yorkshire (Jeet)

## Key Metrics

- Dispensers 4x more efficient (12-15 → 40-45 items/hour)
- 45% reduction in branch stockholding
- Around 50% prescriptions filtered by Titan AI
- Four branches operating as hub-and-spoke network
- Transformation from pharmacies to health clinics
