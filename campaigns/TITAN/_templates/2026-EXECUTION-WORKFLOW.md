# 2026 Content Execution Workflow

## MONTHLY WORKFLOW

### Week 1: Planning & Preparation
- [ ] Review monthly content plan
- [ ] Confirm proof sources available
- [ ] Schedule leadership video filming (if applicable)
- [ ] Identify any missing evidence needed
- [ ] Review canonical coverage targets

### Week 2: Content Creation
- [ ] Create LinkedIn posts (1 week in advance)
- [ ] Write blog (2 weeks before month start)
- [ ] Prepare leadership video script (if applicable)
- [ ] Review quality standards checklist

### Week 3: Review & Refinement
- [ ] Review all content against quality standards
- [ ] Verify proof sources and metrics
- [ ] Check canonical alignment
- [ ] Finalize formats and CTAs

### Week 4: Publishing & Tracking
- [ ] Publish content according to schedule
- [ ] Track performance metrics
- [ ] Update master tracking document
- [ ] Feed performance data back into planning

---

## CONTENT CREATION CHECKLIST

### For Each LinkedIn Post:
- [ ] Named operator OR specific number (or both)
- [ ] Format matches plan (Carousel/Single/Video/Album)
- [ ] Long caption (151+ words for key posts)
- [ ] No abstraction ("a pharmacy owner" forbidden)
- [ ] Maps to canonical idea
- [ ] British English, no hype, no em dashes
- [ ] CTA type matches plan (None/Soft/Hard)
- [ ] Proof sources verified

### For Each Blog:
- [ ] Named operators throughout
- [ ] Specific numbers and metrics
- [ ] Maps to canonical idea
- [ ] Proof sources cited
- [ ] Actionable insights
- [ ] British English, no hype, no em dashes

### For Each Leadership Video:
- [ ] Named operators featured
- [ ] Specific numbers and metrics
- [ ] Hook style matches plan
- [ ] Proof artefacts integrated
- [ ] Clear call to action
- [ ] Maps to canonical idea

## PROOF BANK MANAGEMENT

### Monthly Updates:
- [ ] Review new case studies
- [ ] Update proof bank table
- [ ] Identify missing evidence
- [ ] Plan capture for upcoming events

### Before Each Post:
- [ ] Verify proof source exists

## CANONICAL COVERAGE TRACKING

### Weekly Check:
- [ ] Count posts per canonical
- [ ] Compare to quarterly targets
- [ ] Adjust if needed
- [ ] Update master tracking

### Monthly Review:
- [ ] Review canonical distribution
- [ ] Identify gaps
- [ ] Plan adjustments for next month

---

## FORMAT MIX TRACKING

### Weekly Check:
- [ ] Count posts by format
- [ ] Compare to target distribution
- [ ] Adjust if needed

### Target Distribution:
- Carousels: 60% (93 posts)
- Single Images: 15% (23 posts)
- Video: 20% (31 posts)
- Albums: 5% (8 posts)

---

## PERFORMANCE TRACKING

### Post-Publish:
- [ ] Capture metrics (ER, CTR, TCPS)
- [ ] Update metrics.json files
- [ ] Note top performers
- [ ] Identify underperformers

### Monthly Review:
- [ ] Review performance by format
- [ ] Review performance by canonical
- [ ] Review performance by category
- [ ] Feed insights into next month's planning

## QUALITY STANDARDS ENFORCEMENT

### Pre-Publish Review:

- [ ] Caption length check (key posts)

- [ ] Canonical mapping check
